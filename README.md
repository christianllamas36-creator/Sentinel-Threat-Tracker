# Sentinel Threat Tracker

A command-line threat tracking system built in Python.

## What it does
- Logs threats with ID, type, altitude, and speed
- Automatically classifies each threat as HIGH, MEDIUM, or LOW
- Filters and displays high priority threats
- Saves all threats to a persistent log file
- Loads previous threat logs on launch

## How to run
python threat_tracker.py

## Built with
Python 3.14
